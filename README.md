# Адаптивная версия сайта — Цуканов Вячеслав

## Описание

Создана полностью адаптивная версия сайта, которая корректно отображается на всех типах устройств: десктопах, планшетах и мобильных телефонах.

## Что было сделано

Оригинальный сайт был модифицирован путём добавления адаптивных CSS media queries, которые автоматически изменяют внешний вид сайта в зависимости от размера экрана устройства. Веб-версия и мобильная версия теперь объединены в одном файле.

## Основные улучшения

### Адаптивный дизайн для трёх категорий устройств

**Планшеты и маленькие десктопы (до 1024px):**
- Оптимизированы отступы и размеры шрифтов
- Сохранена горизонтальная компоновка основных элементов
- Улучшена читаемость текста

**Мобильные устройства (до 768px):**
- Header переключается в вертикальную компоновку с центрированием текста
- Кнопки CTA занимают полную ширину экрана и располагаются вертикально
- Таблица тарифов получает горизонтальную прокрутку для удобного просмотра
- Секция автора переключается в колоночную компоновку
- Все изображения масштабируются пропорционально

**Маленькие мобильные устройства (до 480px):**
- Дополнительное уменьшение размеров шрифтов для экономии места
- Оптимизированные отступы между элементами
- Компактное отображение всех секций

### Технические улучшения

- Добавлен `box-sizing: border-box` для всех элементов
- Изображения автоматически масштабируются (`max-width: 100%`)
- Таблицы адаптируются под ширину экрана
- Сохранены все оригинальные функции и ссылки
- Viewport meta-тег уже присутствовал в оригинале

## Файлы

- **index_responsive.html** — финальная адаптивная версия сайта (готова к использованию)
- **index_original.html** — оригинальная версия для сравнения
- **README.md** — данная инструкция
- **test_results.md** — подробные результаты тестирования

## Как использовать

Просто замените ваш текущий `index.html` файлом `index_responsive.html`. Никаких дополнительных настроек не требуется — сайт автоматически адаптируется под размер экрана пользователя.

## Тестирование

Сайт был протестирован на следующих разрешениях:
- Десктоп: 1920x1080
- Планшет: 1024x768
- Мобильный (iPhone 12 Pro): 390x844

Все элементы корректно отображаются и функционируют на всех протестированных устройствах.
