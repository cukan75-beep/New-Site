# Инструкция по установке адаптивной версии сайта

## Быстрый старт

Для внедрения адаптивной версии вашего сайта выполните следующие шаги:

### Шаг 1: Резервное копирование

Перед заменой файла создайте резервную копию вашего текущего сайта. Это позволит вам вернуться к предыдущей версии в случае необходимости.

```bash
cp index.html index_backup.html
```

### Шаг 2: Замена файла

Замените ваш текущий файл `index.html` новым адаптивным файлом `index.html` или `index_responsive.html` из полученного архива.

### Шаг 3: Проверка

Откройте сайт в браузере и проверьте его отображение на разных устройствах. Для тестирования мобильной версии в браузере:

**Google Chrome / Edge:**
1. Откройте сайт
2. Нажмите F12 для открытия DevTools
3. Нажмите Ctrl+Shift+M (или иконку устройства в панели)
4. Выберите различные устройства из выпадающего списка

**Firefox:**
1. Откройте сайт
2. Нажмите F12 для открытия DevTools
3. Нажмите Ctrl+Shift+M
4. Выберите различные устройства

### Шаг 4: Тестирование на реальных устройствах

Для полной уверенности рекомендуется протестировать сайт на реальных мобильных устройствах:
- Откройте сайт на смартфоне
- Проверьте все ссылки и кнопки
- Убедитесь, что текст читаем без горизонтальной прокрутки

## Что изменилось

Новая версия содержит те же самые HTML-элементы и контент, что и оригинал, но с добавленными CSS media queries для адаптивности. Все ваши ссылки, изображения и функциональность остались без изменений.

## Возможные проблемы и решения

### Проблема: Сайт не адаптируется на мобильных

**Решение:** Убедитесь, что в секции `<head>` присутствует следующая строка:
```html
<meta name='viewport' content='width=device-width, initial-scale=1.0'>
```

### Проблема: Некоторые элементы выглядят неправильно

**Решение:** Очистите кеш браузера (Ctrl+F5) и перезагрузите страницу.

### Проблема: Изображения слишком большие на мобильных

**Решение:** Это уже исправлено в новой версии. Если проблема сохраняется, убедитесь, что вы используете правильный файл.

## Дополнительная настройка

Если вам нужно изменить точки перехода (breakpoints) для адаптивности, найдите в CSS следующие media queries и измените значения:

- `@media (max-width: 1024px)` — для планшетов
- `@media (max-width: 768px)` — для мобильных устройств
- `@media (max-width: 480px)` — для маленьких мобильных
